# Arithematic Operators - Magic Methods

## Normal Operations:

```
(I) Assignment Operators:

    (i)   Addition(+):                               __add__(self, other) -------------------- Done(Practised)
    (ii)  Subtraction(-):                            __sub__(self, other) -------------------- Done(Practised)
    (iii) Multiply(*):                               __mul__(self, other)
    (iv)  Divide(/):                                 __truediv__(self, other)
    (v)   Floor Divide(//):                          __floordiv__(self, other)
    (vi)  Modulus(%):                                __mod__(self, other)
    (vii) Exponent(**):                              __pow__(self, other)


(II) BitWise Operators:
    
    (i)   Right Shift(>>):                           __rshift__(self, other)
    (ii)  Left Shift(<<):                            __lshift__(self, other)
    (iii) Bitwise and(&):                            __and__(sel, other)
    (iv)  Bitwse or(|):                              __or__(self, other)
    (v)   Xor(^):                                    __xor__(self, other)

(III) Comparative Operators: --------------- ALL DONE(PRACTISED)
    
    (i)   Greater Than(>):                           __gt__(self, other) --------------------- Done(Practised)
    (ii)  Lesser Than(<):                            __lt__(self, other) --------------------- Done(Practised)
    (iii) Lesser Than Equal to(<=):                  __le__(self, other) --------------------- Done(Practised)
    (iv)  Greater Then Equal to(>=):                 __ge__(self, other) --------------------- Done(Practised)
    (v)   Equal(==):                                 __eq__(self, other) --------------------- Done(Practised)
    (vi)  Not Equal(!=):                             __ne__(self, other) --------------------- Done(Practised)

```

## Incrementers

```
(I) Asignment Operators:

    (i)   Incremented Addition(+=):                  __iadd__(self, other) ------------------- Done(Practised)
    (ii)  Incremented Subtraction(-=):               __isub__(self, other) ------------------- Done(Practised)
    (iii) Incremented Multiply(*=):                  __imul__(self, other)
    (iv)  Incremented Divide(/=):                    __idiv__(self, other)
    (v)   Incremented Floor Divide(//=):             __ifloordiv__(self, other)
    (vi)  Incremented Modulus(%=):                   __imod__(self, other)
    (vii) Incremented Exponent(**=):                 __ipow__(self, other)


(II) BitWise Operators:
    
    (i)   Incremented Right Shift(>>):               __irshift__(self, other)
    (ii)  Incremented Left Shift(<<):                __ilshift__(self, other)
    (iii) Incremented Bitwise and(&):                __iand__(self, other)
    (iv)  Incremented Bitwse or(|):                  __ior__(self, other)
    (v)   Incremented Xor(^):                        __ixor__(self, other)

```
